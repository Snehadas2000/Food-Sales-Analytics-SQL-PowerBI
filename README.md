# Food-Sales-Analytics-SQL-PowerBI
Retail food sales analysis using SQL and Power BI to identify sales trends and product performance.

# Overview
This project analyzes transactional sales data to understand product demand, revenue patterns, and customer purchasing behavior.

# Tools Used
SQL Server
Power BI
Excel

# Key Analysis

• Revenue analysis by product category
• Top selling products
• Peak sales periods
• Order quantity trends
• Category performance

# Insights

• Identified high revenue product categories
• Determined peak sales hours and days
• Analyzed order behavior patterns
